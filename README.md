## 💻 Instalação

### 1. Instalar Node.js
```bash
# Windows (via Chocolatey)
choco install nodejs

# MacOS (via Homebrew)
brew install node

# Linux (Ubuntu/Debian)
curl -fsSL https://deb.nodesource.com/setup_18.x | sudo -E bash -
sudo apt-get install -y nodejs
